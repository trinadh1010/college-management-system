# College Management System – Student Portal

##  Project Description

The College Management System – Student Portal is a simple web-based project developed using HTML, CSS, and JavaScript.
This portal provides a basic interface where students can view important academic and activity-related information such as events, projects, marks, and attendance.

The main purpose of this project is to demonstrate a simple front-end dashboard design for a college management system.

---

##  Features

* Student portal dashboard interface
* View upcoming college events
* Section for student project activities
* Skill improvement resources
* Check academic marks
* Check attendance status
* Clean and responsive user interface

---

##  Student Portal Modules

### 1. Skills Improving

Provides information and activities that help students improve their technical and personal skills.

### 2. Do the Projects

Encourages students to work on practical projects and improve their knowledge.

### 3. Participating in Events

Displays information about college events and allows students to stay updated.

### 4. New Project Launch

Shows newly introduced projects and initiatives.

### 5. Check Your Marks

Allows students to view their academic marks.

### 6. Check Your Attendance

Students can view their attendance information.
